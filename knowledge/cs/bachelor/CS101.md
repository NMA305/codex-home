# CS101: Introduction to Programming

## مقدمة إلى البرمجة (CS101 – Introduction to Programming)

### 1. ما هي البرمجة؟
البرمجة هي عملية **كتابة تعليمات** (أوامر) تُعطى للكمبيوتر ليقوم بأداء مهمة معينة.  
هذه التعليمات تُكتب بلغة **برمجية** تُترجم أو تُفسّر إلى لغة الآلة التي يفهمها الحاسوب.

> **تشبيه:**  
> إذا كان الكمبيوتر “طباخًا” لا يعرف إلا وصفات (الرموز الثنائية)، فإن البرمجة هي كتابة **وصفة** (البرنامج) بلغة يستطيع الطباخ قراءتها وتنفيذها.

---

## 2. عناصر البرنامج الأساسي

| العنصر | الوصف | مثال (بايثون) |
|--------|-------|----------------|
| **المتغيّر (Variable)** | مساحة تخزين تحمل قيمة يمكن تغييرها أثناء تشغيل البرنامج. | `age = 25` |
| **النوع (Data Type)** | يحدد طبيعة البيانات (عدد صحيح، نص، منطقية …). | `int`, `float`, `str`, `bool` |
| **التعبير (Expression)** | عملية حسابية أو منطقية تُنتج قيمة. | `a + b * 2` |
| **الجملة (Statement)** | أمر يُنفَّذ (إسناد، طباعة، شرط …). | `print("Hello")` |
| **الدالة (Function)** | مجموعة من الأوامر تُستدعى باسم واحد لتكرار استخدامها. | `def greet(name):` |
| **التحكم (Control Flow)** | يحدد أي جزء من البرنامج يُنفَّذ (شرط، حلقة …). | `if`, `for`, `while` |

---

## 3. بيئة التطوير (IDE / Interpreter)

| الأداة | الوصف | لماذا تُستَخدم |
|--------|-------|----------------|
| **Python Interpreter** | يقرأ أوامر بايثون سطرًا بسطر. | سهل التعلم، متعدد المنصات. |
| **IDLE** | محرر نصوص بسيط مدمج مع بايثون. | للمبتدئين. |
| **VS Code / PyCharm** | محررات متقدمة تدعم الإكمال التلقائي، تصحيح الأخطاء، وإدارة المشاريع. | لتطوير مشاريع أكبر. |

> **ملاحظة:** في هذا الدرس سنستخدم **بايثون** لأنها لغة عالية المستوى، صديقة للمبتدئين، وتدعم جميع المفاهيم الأساسية للبرمجة.

## 4. برنامج “Hello, World!” – أول خطوة

```python
# برنامج بسيط يطبع جملة على الشاشة
print("Hello, World!")
```

* **`print`**: دالة مدمجة تُظهر النص على الشاشة.  
* النص بين علامتي اقتباس مزدوجتين (`" "` ) هو **سلسلة نصية** (String).  

> **تشغيل البرنامج**  
> 1. احفظ الملف باسم `hello.py`.  
> 2. افتح الطرفية (Terminal) واكتب: `python hello.py`.  
> 3. ستظهر لك الجملة `Hello, World!`.

## 5. المتغيّرات والأنواع (Variables & Data Types)

```python
# تعريف متغيّرات من أنواع مختلفة
age = 21               # عدد صحيح (int)
height = 1.78          # عدد عشري (float)
name = "Layla"         # نص (str)
is_student = True      # قيمة منطقية (bool)

print(name, "is", age, "years old.")
```

* **الأنواع الأساسية**  
  - `int` : أعداد صحيحة (مثال: `-3`, `0`, `42`).  
  - `float` : أعداد ذات فاصلة عشرية (مثال: `3.14`).  
  - `str` : سلاسل نصية (مثال: `"Hello"`).  
  - `bool` : قيم منطقية (`True` أو `False`).  

* **قواعد تسمية المتغيّرات**  
  - تبدأ بحرف أو underscore (`_`).  
  - لا يمكن أن تحتوي على مسافات أو رموز خاصة (باستثناء `_`).  
  - حساسة لحالة الأحرف (`age` ≠ `Age`).

## 6. العمليات (Operators)

| الفئة | المشغّل | مثال | النتيجة |
|-------|----------|------|----------|
| **حسابية** | `+` `-` `*` `/` `//` `%` `**` | `5 + 3` | `8` |
| **مقارنة** | `==` `!=` `<` `>` `<=` `>=` | `5 > 2` | `True` |
| **منطقية** | `and` `or` `not` | `True and False` | `False` |
| **تخصيص** | `=` `+=` `-=` `*=` `/=` | `x += 2` (يعني `x = x + 2`) | — |

```python
a = 10
b = 3

print(a + b)   # جمع
print(a // b)  # قسمة صحيحة (نتيجة عدد صحيح)
print(a % b)   # باقي القسمة
print(a > b)   # مقارنة
print((a > b) and (b > 0))  # منطقية
```

## 7. التحكم في تدفق البرنامج

### 7.1 الجمل الشرطية (if‑elif‑else)

```python
score = 78

if score >= 90:
    print("A")
elif score >= 80:
    print("B")
elif score >= 70:
    print("C")
else:
    print("D أو أقل")
```

* **المسافات البادئة (Indentation)** في بايثون ضرورية لتحديد نطاق الجملة.

### 7.2 الحلقات (Loops)

#### 7.2.1 حلقة `for` (تكرار على مجموعة)

```python
# طباعة الأعداد من 1 إلى 5
for i in range(1, 6):      # range يبدأ من 1 وينتهي قبل 6
    print(i)
```

#### 7.2.2 حلقة `while` (تكرار بناءً على شرط)

```python
count = 0
while count < 3:
    print("Iteration:", count)
    count += 1            # زيادة المتغيّر لتفادي حلقة لا نهائية
```

#### 7.2.3 كسر الحلقة (break) واستمرارها (continue)

```python
for n in range(1, 11):
    if n == 5:
        continue          # تخطى طباعة 5
    if n == 8:
        break             # أوقف الحلقة عند 8
    print(n)
```

## 8. الدوال (Functions)

الدالة هي **كتلة من التعليمات** يمكن استدعاؤها بأسماء ومعطيات (parameters) لتعيد **قيمة** (return value) أو تُنفّذ عملًا.

```python
def greet(name):
    """ترحب بالمستخدم."""
    return f"مرحبًا، {name}!"

# استدعاء الدالة
message = greet("سليم")
print(message)   # يطبع: مرحبًا، سليم!
```

### 8.1 معطيات اختيارية (Default Arguments)

```python
def power(base, exponent=2):
    return base ** exponent

print(power(5))      # 25 (5²)
print(power(5, 3))   # 125 (5³)
```

### 8.2 القيم المرجعية vs القيم القابلة للنسخ

- الأنواع القابلة للنسخ (immutable) مثل `int`, `float`, `str`, `tuple` تُرسل كقيمة.
- الأنواع القابلة للتغيير (mutable) مثل `list`, `dict` تُرسل كمرجع؛ تعديلها داخل الدالة ينعكس خارجها.

```python
def add_one(lst):
    lst.append(1)   # تعديل القائمة مباشرة

my_list = [0, 0]
add_one(my_list)
print(my_list)    # [0, 0, 1]
```

## 9. قراءة وكتابة الملفات (File I/O)

```python
# كتابة ملف نصي
with open("data.txt", "w") as f:   # وضع 'w' يعني كتابة (تستبدل إذا كان موجودًا)
    f.write("سطر أول\n")
    f.write("سطر ثاني\n")

# قراءة الملف
with open("data.txt", "r") as f:
    for line in f:
        print(line.strip())   # strip لإزالة \n
```

* **`with`** يضمن إغلاق الملف تلقائيًا حتى لو حدث استثناء.

## 10. أخطاء شائعة للمبتدئين وكيفية تجنّبها

| الخطأ | السبب | الحل |
|-------|-------|------|
| **SyntaxError** | كتابة غير صحيحة (نقطة فاصلة، أقواس مفقودة) | تحقق من الأقواس والمسافات البادئة. |
| **NameError** | استدعاء متغيّر أو دالة غير معرفة | تأكد من تعريف المتغيّر قبل الاستخدام. |
| **TypeError** | محاولة جمع نوعين غير متوافقين (`int + str`) | حول الأنواع باستخدام `int()`, `str()`, `float()` إذا لزم. |
| **IndexError** | الوصول إلى عنصر خارج نطاق القائمة | افحص طول القائمة (`len(list)`). |
| **Infinite Loop** | عدم تعديل المتغيّر داخل `while` | تأكد من وجود شرط يغيّر قيمة المتغيّر. |

## 11. مشروع صغير لتطبيق ما تعلمته

### هدف المشروع  
إنشاء **برنامج لإدارة مكتبة صغيرة** يتيح للمستخدم:
1. إضافة كتاب (عنوان، مؤلف، سنة النشر).  
2. عرض جميع الكتب.  
3. البحث عن كتاب بحسب العنوان.  
4. حفظ البيانات إلى ملف `books.json` وتحميلها عند تشغيل البرنامج.

### هيكل البرنامج (ملخص)

```python
import json

FILE = "books.json"

def load_books():
    try:
        with open(FILE, "r") as f:
            return json.load(f)
    except FileNotFoundError:
        return []        # لا يوجد ملف → قائمة فارغة

def save_books(books):
    with open(FILE, "w") as f:
        json.dump(books, f, ensure_ascii=False, indent=2)

def add_book(books):
    title = input("العنوان: ")
    author = input("المؤلف: ")
    year = input("سنة النشر: ")
    books.append({"title": title, "author": author, "year": year})
    print("تم إضافة الكتاب.")

def list_books(books):
    if not books:
        print("لا توجد كتب.")
        return
    for i, b in enumerate(books, 1):
        print(f"{i}. {b['title']} - {b['author']} ({b['year']})")

def search_book(books):
    query = input("ابحث بالعنوان: ").lower()
    results = [b for b in books if query in b["title"].lower()]
    if results:
        for b in results:
            print(f"{b['title']} - {b['author']} ({b['year']})")
    else:
        print("لم يُعثر على أي كتاب.")

def main():
    books = load_books()
    while True:
        print("\n--- مكتبة الكتب ---")
        print("1. إضافة كتاب")
        print("2. عرض الكتب")
        print("3. بحث")
        print("4. حفظ & خروج")
        choice = input("اختر رقمًا: ")

        if choice == "1":
            add_book(books)
        elif choice == "2":
            list_books(books)
        elif choice == "3":
            search_book(books)
        elif choice == "4":
            save_books(books)
            print("تم الحفظ. وداعًا!")
            break
        else:
            print("اختيار غير صالح. أعد المحاولة.")

if __name__ == "__main__":
    main()
```

**ما تعلمته من خلال هذا المشروع:**
- تعريف واستخدام القوائم (`list`) والقواميس (`dict`).  
- قراءة/كتابة ملفات JSON (تنسيق شائع لتخزين البيانات).  
- هيكلية برنامج تفاعلي باستخدام الحلقات والشروط.  
- تقسيم البرنامج إلى **دوال** لتقليل التكرار وتحسين القابلية للصيانة.
